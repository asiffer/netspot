Download

netspot is available on Linux platforms with different architectures: amd64, arm and aarch64. The Go ecosystem would allow to easily build netspot for Mac and Windows however it has not been tested, and that is probably not very relevant.

So the easiest way to get netspot is to download the latest pre-built binary corresponding to your architecture.

curl -Lo netspot 'https://github.com/asiffer/netspot/releases/download/v2.0a/netspot-2.0a-amd64-linux-static'
chmod +x netspot

Let us recall that you need some privilege to run netspot on network interfaces. So either you can run it as root (not recommended) or you can add the sniffing capability.

sudo setcap cap_net_admin,cap_net_raw=eip ./netspot
