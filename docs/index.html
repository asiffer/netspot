netspot

A simple IDS with statistical learning

Overview Get started

Smart

Netspot finds abnormal behaviors in network statistics. Based on the SPOT algorithm, it automatically builds and updates a threshold aimed to flag extreme events (peaks).

Fast

Netspot is basically able to treat 1M packets/s on the guy next door computer. Even on embedded device like a Raspberry Pi (3B+), netspot can process about 100k packets/s.

Modular

Netspot get used to your needs. Developers can easily implement new statistics and new connectors to send netspot data where they want.
